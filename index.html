Stateless.js

Lightweight cross-browser context persistence library Ã  la pushState, via changing the hash fragment. Mainly meant for freeloading serverless web applications which need to store some state in the URL, and provides some nice conveniences like:

  • Preventing repeated processing of consecutive, identical hashes.
  • Ability to mark hashes as "processed".
// the hash will always be #nope
Stateless.onChange(function(fragment) {
  Stateless.push('nope');
});
Stateless.pull();
Stateless.skip('processed');
